<CardGrid>
  <Card title="対象ドメイン限定" icon="approve-check">

  </Card>
  <Card title="HTTPS MITM" icon="seti:lock">
    自分専用CAでTLSを復号し、TLS内のリクエストにもヘッダーを追加します。
  </Card>
  <Card title="安全側の既定" icon="warning">
    機密ヘッダーのログ自動マスク、クライアントIP許可リスト、既定10分で自動停止。
  </Card>
  <Card title="クリーンな構成" icon="setting">
    Go製のシンプルなCLI。`--gen-ca` で自分専用CAも生成できます。
  </Card>
